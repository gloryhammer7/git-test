# Hello Lilia Go App

Простое веб-приложение на Go для тренировки Git и Docker.

## Ветки

- **main** - основная ветка
- **story-1** - альтернативная ветка с другим текстом

## Как запустить

### С использованием Docker

1. Соберите образ:
   ```bash
   docker build -t hello-lilia .
   ```

2. Запустите контейнер:
   ```bash
   docker run -p 8080:8080 hello-lilia
   ```

3. Откройте в браузере: http://localhost:8080

### Без Docker (требуется Go)

1. Запустите приложение:
   ```bash
   go run main.go
   ```

2. Откройте в браузере: http://localhost:8080

## Упражнения для новичка

1. Склонируйте репозиторий
2. Переключитесь на ветку `story-1` и посмотрите отличия
3. Соберите Docker-образ из обеих веток и проверьте разный текст
4. Попробуйте смержить ветку story-1 в main
